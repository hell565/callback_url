# Suno Music Telegram Bot

Telegram-бот для генерации музыки с помощью **Suno API** (неофициальный sunoapi.org).

Позволяет пользователям создавать песни по текстовому описанию — от простых промптов до продвинутого режима с собственными текстами, стилем, моделью и инструменталом.

## Что умеет бот

- **Простой режим**  
  Пишешь описание песни → бот генерирует 1–2 варианта трека автоматически (вокал + музыка по умолчанию)

- **Продвинутый режим** (в разработке / частично)  
  - Собственный текст песни (lyrics)  
  - Указание стиля/жанра/тегов  
  - Название трека  
  - Instrumental (без вокала)  
  - Выбор модели (V5, V4_5ALL, V4_5PLUS и др.)  
  - Продление существующего трека (extend) — скоро

- После генерации бот присылает:  
  - Обложку трека (картинка)  
  - Название песни  
  - Длительность  
  - Кусок текста (если есть)  
  - Кнопки: скачать MP3, новая песня, продлить трек

- Работает асинхронно через **callback** от Suno API  
  → сервер скачивает треки автоматически → бот отправляет их пользователю
